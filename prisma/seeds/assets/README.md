# Seed public assets

Local source files for the shared public assets bucket
(`AWS_S3_PUBLIC_ASSETS_BUCKET`), served via CloudFront
(`AWS_S3_PUBLIC_ASSETS_BASE_URL`).

Organize by feature. Persona schema UI assets live under:

```text
persona/schema-v1/
```

Add more top-level folders later (e.g. `marketing/`, `onboarding/`) as needed.
